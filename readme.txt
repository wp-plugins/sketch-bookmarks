== Description ==

This plugin contains amazing looking, sketched icons for only the top social bookmarking sites. Quit confusing your readers with dozens of bookmarking options and go start to really go after those that or actually getting used with these amazing looking sketch icons.

== Installation ==

1. Download sketch-bookmarks.zip
2. Extract contents of sketch-bookmarks.zip creating a folder called sketch-bookmarks.
3. Upload the newly created sketch bookmarks folder to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Adjust the settings as necessary under Settings --> Sketch Bookmarks
